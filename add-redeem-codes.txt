## Method 2: Permanent (Edit js/script.js)
1. Open js/script.js in a text editor
2. Find line ~170 (the "Add your new codes here" section)
3. Uncomment a line by removing the "//" at the beginning
4. Replace with your code:

addRedeemCode('YOUR_NEW_CODE', 'Description of rewards');

Example:
addRedeemCode('FCMOBILE2025', 'New Year Special - Get 1000 gems!');

5. Save the file
6. Refresh your website

## Method 3: Replace Existing Code (Edit index.html)
1. Open index.html in text editor
2. Find the existing code around line 130:
   <span class="code-text">FCMOBILE2024</span>
3. Replace "FCMOBILE2024" with your new code
4. Update the description below it
5. Save and refresh

## Tips:
- Always test the copy functionality after adding codes
- Keep descriptions short and clear
- Use ALL CAPS for codes (standard format)
- Update codes regularly to keep visitors engaged
